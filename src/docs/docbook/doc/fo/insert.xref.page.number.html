Name

insert.xref.page.number — Turns page numbers in xrefs on and off

Synopsis

§160: §199

  1| <xsl:param name="insert.xref.page.number"
  2|            select="0"/>

Description

If non-zero, cross references (xrefs) will include page number citations.
