[#rnfb60-msql-search-path]
== `SET SEARCH_PATH TO`
Adriano dos Santos Fernandes

The `SET SEARCH_PATH TO` changes the <<rnfb60-engine-schema-path,search path>> for the current session.

.Syntax
[listing, subs=+quotes]
----
SET SEARCH_PATH TO _schema_ [, _schema_]...
----

If `SYSTEM` is not included in the schema list, it's automatically added as the last schema in the search path.
Executing `ALTER SESSION RESET` reverts to the initial search path as configured with `isc_dpb_search_path` or `PUBLIC, SYSTEM`.

.Example
[source]
----
set search_path to CUSTOMERS, ORDERS;
-- Changes search path to CUSTOMERS, ORDERS, SYSTEM
----
