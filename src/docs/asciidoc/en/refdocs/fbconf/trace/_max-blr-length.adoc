[#fbconf-trace-max-blr-length]
= `max_blr_length`

Maximum length of BLR (Binary Language Representation) in a log record.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
max_blr_length = _integer_
----

.Unit
Byte

.Default
`500`

Beware when adjusting `max++_++__XXX__` parameters!
The maximum length of a log record for one event should not exceed 64KiB.

.See also
<<fbconf-trace-print-blr>>
