[#fbconf-trace-max-arg-count]
= `max_arg_count`

Maximum number of query arguments in a log record.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
max_arg_count = _integer_
----

.Unit
Argument (parameter)

.Default
`30`

Beware when adjusting `max++_++__XXX__` parameters!
The maximum length of a log record for one event should not exceed 64KiB.

.See also
<<fbconf-trace-log-statement-start>>, <<fbconf-trace-max-arg-length>>
