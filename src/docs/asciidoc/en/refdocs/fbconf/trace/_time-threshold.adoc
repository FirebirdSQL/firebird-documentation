[#fbconf-trace-time-threshold]
= `time_threshold`

Only logs `__XXX__++_++finish` events if their timing exceeds the threshold.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
time_threshold = _integer_
----

.Unit
Millisecond

.Default
`100` (100 milliseconds)

.See also
<<fbconf-trace-log-statement-finish>>, <<fbconf-trace-log-procedure-finish>>, <<fbconf-trace-log-function-finish>>, <<fbconf-trace-log-trigger-finish>>
