[#fbconf-trace-print-plan]
= `print_plan`

Print access path (plan) with SQL statement.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
print_plan = _Boolean_
----

.Default
`false`

The type of plan -- legacy or explained -- can be controlled with <<fbconf-trace-explain-plan>>.
