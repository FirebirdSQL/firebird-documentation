[#fbconf-trace-max-sql-length]
= `max_sql_length`

Maximum length of SQL strings in a log record.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
max_sql_length = _integer_
----

.Unit
Byte

.Default
`300`

Beware when adjusting `max++_++__XXX__` parameters!
The maximum length of a log record for one event should not exceed 64KiB.

.See also
<<fbconf-trace-log-statement-prepare>>
