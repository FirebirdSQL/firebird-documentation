[#fbconf-trace-max-arg-length]
= `max_arg_length`

Maximum length of an individual string argument in a log record.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
max_arg_length = _integer_
----

.Unit
Byte

.Default
`80`

Beware when adjusting `max++_++__XXX__` parameters!
The maximum length of a log record for one event should not exceed 64KiB.

.See also
<<fbconf-trace-log-statement-start>>, <<fbconf-trace-max-arg-count>>
