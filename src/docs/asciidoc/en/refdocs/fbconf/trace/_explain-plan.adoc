[#fbconf-trace-explain-plan]
= `explain_plan`

Print legacy plan (`false`) or explained plan (`true`).

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
explain_plan = _Boolean_
----

.Default
`false` (legacy plan)

This setting only has effect when <<fbconf-trace-print-plan>> is enabled.
