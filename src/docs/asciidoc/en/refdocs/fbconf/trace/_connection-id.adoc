[#fbconf-trace-connection-id]
= `connection_id`

Limits the trace to a specific connection.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
connection_id = _integer_
----

.Default
`0` (all connections)

The value `0` (zero, the default) traces all connections.
A non-zero value traces the connection with that specific connection id.

The connection id is the value reported by -- for example -- https://firebirdsql.org/file/documentation/chunk/en/refdocs/fblangref50/fblangref50-contextvars.html#fblangref50-contextvars-current-connection[`CURRENT_CONNECTION`], and in column `MON$ATTACHMENT_ID` of https://firebirdsql.org/file/documentation/chunk/en/refdocs/fblangref50/fblangref50-appx05-montables.html#fblangref-appx05-monattach[`MON$ATTACHMENTS`].

It's not possible to specify multiple connection ids.
