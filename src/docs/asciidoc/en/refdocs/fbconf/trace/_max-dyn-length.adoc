[#fbconf-trace-max-dyn-length]
= `max_dyn_length`

Maximum length of dyn in a log record.

.Trace type
`database`

.Syntax
[listing,subs=+quotes]
----
max_dyn_length = _integer_
----

.Unit
Byte

.Default
`500`

Beware when adjusting `max++_++__XXX__` parameters!
The maximum length of a log record for one event should not exceed 64KiB.

.See also
<<fbconf-trace-print-dyn>>
