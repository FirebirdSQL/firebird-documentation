[#fbconf-repl-verbose-logging]
= `verbose_logging`

Configures if `replication.log` contains a detailed log of operations performed by replication.

.Replication type
Asynchronous

// TODO Check if also applies to synchronous

.Syntax
[listing,subs=+quotes]
----
verbose_logging = _Boolean_
----

.Default
`false`

If disabled (the default), only errors and warnings are logged.

This setting only affects logging on the replica side.
For error logging on the primary side, see <<fbconf-repl-log-errors>>.

.See also
<<fbconf-repl-log-errors>>
