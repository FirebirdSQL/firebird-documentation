[#fbconf-repl-log-errors]
= `log_errors`

Configures if all errors and warnings are logged to `replication.log`.

.Replication type
Synchronous, asynchronous

.Availability
[horizontal.compact]
Added:: 4.0

.Syntax
[listings,subs=+quotes]
----
log_errors = _Boolean_
----

.Default
`true`

This setting only affects error logging on the primary side.
For error logging on the replica side, see <<fbconf-repl-verbose-logging>>.

.See also
<<fbconf-repl-verbose-logging>>
