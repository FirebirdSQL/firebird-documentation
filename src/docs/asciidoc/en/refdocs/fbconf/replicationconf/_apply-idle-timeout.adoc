[#fbconf-repl-apply-idle-timeout]
= `apply_idle_timeout`

Timeout (in seconds) to wait before scanning for new replication segments.

.Replication type
Asynchronous

.Availability
[horizontal.compact]
Added:: 4.0

.Syntax
[listing,subs=+quotes]
----
apply_idle_timeout = _integer_
----

.Unit
Second

.Default
`10` (10 seconds)

When all existing journal files have been applied, replication waits for the specified duration before checking if new files have arrived in <<fbconf-repl-journal-source-directory>>.

.See also
<<fbconf-repl-journal-source-directory>>
