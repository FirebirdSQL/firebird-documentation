[#fbconf-repl-plugin]
= `plugin`

Plugin used to perform replication.

.Replication type
Synchronous, asynchronous

.Availability
[horizontal.compact]
Added:: 4.0

.Syntax
[listing,subs=+quotes]
----
plugin = _plugin_
----

.Default
_empty_ (use built-in replication)

Leaving/setting this empty will use the built-in (default) replication.

.Example
[listing]
----
# Use plugin CustomReplication for all databases
database
{
  plugin = CustomReplication
}

# Revert to default replication for a specific database
database /path/to/example.fdb
{
  plugin =
}
----

[NOTE]
====
The `plugin` configuration item only affects the primary side of replication.
The replica side cannot be overridden by specifying an alternative plugin.
====
