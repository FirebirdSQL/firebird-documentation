[#fbconf-repl-report-errors]
= `report_errors`

Configures if replication errors are reported to the client application.

.Replication type
Synchronous, asynchronous

.Availability
[horizontal.compact]
Added:: 4.0

.Syntax
[listings,subs=+quotes]
----
report_errors = _Boolean_
----

.Default
`false`
