[#fbconf-repl-journal-segment-size]
= `journal_segment_size`

Maximum allowed size for a single replication segment.

.Replication type
Asynchronous

.Availability
[horizontal.compact]
Added:: 4.0

.Syntax
[listing,subs=+quotes]
----
journal_segment_size = _integer_
----

.Unit
Byte

.Default
`16777216` (16 MiB)

.See also
<<fbconf-repl-journal-archive-timeout>>
