[[fblangref40-ddl-de]]
= Data Definition (DDL)-Anweisungen

DDL ist die Teilmenge der Datendefinitionssprache der SQL-Sprache von Firebird.
DDL-Anweisungen werden verwendet, um von Benutzern erstellte Datenbankobjekte zu erstellen, zu ändern und zu löschen.
Wenn eine DDL-Anweisung festgeschrieben wird, werden die Metadaten für das Objekt erstellt, geändert oder gelöscht.
